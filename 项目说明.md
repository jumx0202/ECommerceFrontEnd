# 电子商务多渠道销售与库存管理一体化系统 - 前端界面

## 项目概述

本项目为电子商务多渠道销售与库存管理一体化系统的Vue3前端界面，基于操作手册要求设计，提供美观现代的用户界面和完整的业务功能。

## 技术栈

- **前端框架**: Vue 3 + TypeScript
- **UI组件库**: Element Plus 2.9.11
- **路由管理**: Vue Router 4
- **状态管理**: Pinia + 持久化插件
- **图表组件**: ECharts + Vue ECharts
- **HTTP客户端**: Axios
- **构建工具**: Vite
- **动画库**: Animate.css
- **工具库**: @vueuse/core
